# 🖱️ Guide d'utilisation - Positionnement interactif de la signature

## ✨ Nouvelle fonctionnalité : Drag & Drop

Vous pouvez maintenant **voir et positionner visuellement** votre signature directement sur le document !

---

## 📖 Comment utiliser

### Étape 1 : Télécharger votre document
1. Cliquez sur "Choisir un fichier"
2. Sélectionnez un PDF ou une image

### Étape 2 : Créer votre signature
1. Dessinez votre signature dans la zone prévue
2. Choisissez la couleur et l'épaisseur

### Étape 3 : Positionner la signature ⭐ NOUVEAU !

Vous avez **2 options** pour positionner votre signature :

#### Option 1 : Glisser-Déposer (Drag & Drop) 🖱️

1. Cliquez sur le bouton **"👁️ Afficher la signature sur le document"**
2. Votre signature apparaît sur le document avec une bordure en pointillés
3. **Cliquez et maintenez** la signature
4. **Glissez-la** où vous voulez sur le document
5. **Relâchez** pour la placer
6. Les coordonnées X et Y se mettent à jour automatiquement !

#### Option 2 : Contrôles précis 🎯

Utilisez les champs numériques pour un placement exact :
- **Position X** : Position horizontale (0-595 pixels)
- **Position Y** : Position verticale (0-842 pixels)
- **Largeur** : Taille de la signature (50-300 pixels)

---

## 💡 Astuces

### Sur ordinateur 🖥️
- **Survol** : La signature change de couleur quand vous passez la souris dessus
- **Clic maintenu** : Le curseur devient une main qui attrape
- **Déplacement** : Glissez librement, la signature ne sort pas du document

### Sur mobile/tablette 📱
- **Toucher** : Appuyez sur la signature
- **Glisser** : Déplacez votre doigt
- **Relâcher** : La signature reste en place

### Ajustements 🎨
- Utilisez le **slider de largeur** pour agrandir ou rétrécir la signature
- La valeur s'affiche en temps réel (ex: "150px")
- La signature sur le document se redimensionne instantanément

## 🐛 Résolution de problèmes

### La signature ne s'affiche pas
→ Vérifiez que vous avez bien dessiné une signature d'abord

### La signature ne bouge pas
→ Assurez-vous d'avoir cliqué sur "Afficher la signature"

### La position n'est pas précise
→ Utilisez les champs numériques X et Y pour un placement exact

### La signature est trop grande/petite
→ Ajustez le slider "Largeur" entre 50 et 300 pixels
