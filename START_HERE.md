## ⚡ EN BREF

1. **Supabase** : Exécutez `supabase/schema.sql` dans SQL Editor
2. **Storage** : Créez le bucket `signed-documents` (public)
3. **Vercel** : Copiez les 8 variables de `VERCEL_ENV_READY.txt`
4. **Deploy** : Cliquez sur "Redeploy"
5. **URL** : Ajoutez votre URL Vercel dans `NEXT_PUBLIC_APP_URL`
6. **Redeploy** : Une dernière fois
